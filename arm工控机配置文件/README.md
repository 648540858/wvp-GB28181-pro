
### Nginx配置
> 端口检查sudo netstat -ap | grep 80
#### 相关路径

安装路径:`/usr/loca/nginx`  
配置文件路径: `/usr/local/nginx/conf/nginx.conf`

### Java配置
>版本查看 java -version
#### 相关路径
> jdk1.8    

安装路径: `/usr/local/java`

### Redis配置
> 使用默认配置    
> 端口检查sudo netstat -ap | grep 80

### 相关路径
安装位置：`/usr/local/redis`
